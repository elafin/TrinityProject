**Opis projektu**

Aplikacja webowa umożliwiająca automatyczne generowanie diagramu klas UML na podstawie:
* lokalnych plików .class / .java
* przesłanych archiwów .jar

Projekt wykorzystuje mechanizmy refleksji w języku Java, aby analizować strukturę klas i przedstawiać ją w czytelnej, graficznej formie.

**Cel:**
 - Analiza klas z plików lokalnych
 - Analiza klas z przesłanych plików JAR
 - Odczyt nazw klas, pól, metod, modyfikatorów dostępu, typów parametrów i wartości zwracanych
 - Automatyczne generowanie diagramu

**Wykorzystane technologie:**
 - Java 21
 - Spring Boot 3.5.7
 - Thymeleaf
 - HTML5
 - CSS3
 - JavaScript
 - MariaDB 10.4.32
